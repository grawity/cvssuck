Download CVSsuck

with HTTP to get latest tarball:

CVSsuck-latest.tar.gz

with CVS:
% cvs -d :pserver:anonymous@cvs.m17n.org:/cvs/cvs login
(Logging in to anonymous@cvs.m17n.org)
CVS password: (just hit return)
% cvs -d :pserver:anonymous@cvs.m17n.org:/cvs/cvs co cvssuck
The cvs password can be any string.
with CVSup:
% cvsup -g supfile.cvssuck
with CVSsuck:
% cvssuck :pserver:anonymous@cvs.m17n.org:/cvs/cvs cvssuck
